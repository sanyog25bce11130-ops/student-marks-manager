# student-marks-manager

This is a simple Python project made using very basic Python concepts such as:
- Lists
- Loops
- If-else conditions
- Functions
- User input

The project allows you to:
- Add student names and marks
- Show all students
- Search a student
- Find highest marks
- Calculate class average

## How to Run
1. Install Python 3
2. Download the project files
3. Run the command:
   python project.py
